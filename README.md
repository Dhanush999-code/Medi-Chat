# MediChat AI

An AI-powered healthcare assistant built using Flask and Google Gemini AI.

## Features
- AI Medical Chatbot
- Medicine Analysis from Images
- Bill Verification
- Complaint Management
- Emergency Contacts
- Chat History

## Tech Stack
- Python Flask
- SQLite
- HTML/CSS/JavaScript
- Google Gemini API

## Setup

pip install -r requirements.txt
python app.py
